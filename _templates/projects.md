# Projects

Local index of all projects in this Foreman workspace. This file is **git-ignored** — it is your private state and is never committed to the framework repo. Updated by `/new-project` and `/absorb` after sign-off, and manually when status changes.

---

## Visibility

- `public` — pushed to a public GitHub repo; anyone can clone or use it
- `private` — own private git repo; never exposed through Foreman

## Status values

- `active` — in development
- `paused` — on hold, may resume
- `complete` — all M3 milestone criteria met, shipped and stable; no active work planned
- `abandoned` — cancelled; add a note in the Goal column explaining why
